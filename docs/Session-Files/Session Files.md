# Session Files 

## Downloading Session (*.cys) Files

<strong>IMPORTANT:</strong> To save a file in Cytoscape format (*.cys) to your local drive, <strong>DO NOT</strong> right-click it and select <strong>Save Link As ...</strong>.  Instead, click the network name, and on the new page, select either <strong>Raw</strong> button or <strong>View Raw</strong> link.  A confirmation dialogue may pop up, select <strong>Save File</strong> button.
